# Color selector

| Action                                        | Result                                                     |
| --------------------------------------------- | ---------------------------------------------------------- |
| Hover over **Edit icon**                      | Preview animation plays showing the edit action            |
| Click **Edit icon** with a color selected     | Opens edit mode for the selected color and plays animation |
| Click **Edit icon** without a color selected  | No action                                                  |
| Hover over **Trash icon**                     | Preview animation plays showing the delete action          |
| Click **Trash icon** with a color selected    | Deletes the selected color and plays animation             |
| Click **Trash icon** without a color selected | No action                                                  |
